Overview

Tasks help you manage follow-ups, to-dos, and action items. Tasks can be linked to customers, leads, or deals to keep everything organized.

ü§ñ Automatic Task Creation

The system automatically creates follow-up tasks for leads after 7 days. These tasks are assigned to the lead owner (SDR) to ensure timely follow-up.

Task Types

  • Call: Phone call to make
  • Email: Email to send
  • Meeting: Meeting to schedule
  • General: General task
  • Follow-up: Follow-up task (auto-generated)

How to Create a Task

1 Navigate to Tasks Page

Click on "Tasks" in the sidebar navigation menu.

2 Click "Add Task" Button

Click the "Add Task" button at the top right.

3 Fill in Task Information

Complete the task form:

  • Title: Task title (required)
  • Description: Detailed description
  • Task Type: Call, Email, Meeting, General, Follow-up
  • Priority: Low, Medium, High, Urgent
  • Due Date: When the task should be completed
  • Assigned To: Who is responsible
  • Related To: Link to Customer, Lead, or Deal (optional)
4 Save Task

Click "Save" to create the task. It will appear in your task list.

Task Status

  • Pending: Task not started
  • In Progress: Task in progress
  • Completed: Task finished
  • Cancelled: Task cancelled

Managing Tasks

1 View Tasks

The tasks page shows all tasks with:

  • Task title
  • Type and priority
  • Due date
  • Status
  • Assigned owner
  • Related entity (Customer/Lead/Deal)
2 Filter Tasks

Filter by:

  • Status (Pending, In Progress, Completed)
  • Priority (Low, Medium, High, Urgent)
  • Task Type
  • Assigned To
3 Complete Task

Click "Complete" or edit the task and change status to "Completed". The system will record the completion date.

Priority Levels

  • Low: Can be done when convenient
  • Medium: Normal priority
  • High: Important, should be done soon
  • Urgent: Critical, needs immediate attention
